# goit-rdb-hw-07

Домашнє завдання до теми 7: Додаткові вбудовані SQL-функції. Робота з часом.

---

## Overview

У цьому проєкті реалізовано:

- роботу з датою та часом у SQL
- використання вбудованих функцій обробки дати
- перетворення дати у timestamp
- фільтрацію даних за часовим діапазоном
- формування JSON-структур безпосередньо в SQL
---

## Task 1 — Extract YEAR, MONTH, DAY

Реалізовано витяг компонентів дати:
- рік → `YEAR(date)`
- місяць → `MONTH(date)`
- день → `DAY(date)`

Вивід:
- `id`
- `date`
- `order_year`
- `order_month`
- `order_day`

---

## Task 2 — Add 1 Day

Реалізовано додавання 1 дня до дати:

- використано `DATE_ADD(date, INTERVAL 1 DAY)`

Вивід:
- `id`
- `date`
- `date_plus_1_day`

---

## Task 3 — Timestamp

Реалізовано перетворення дати у timestamp:

- використано `UNIX_TIMESTAMP(date)`

Вивід:
- `id`
- `date`
- `date_timestamp`

---

## Task 4 — Count in Date Range

Реалізовано підрахунок записів у часовому діапазоні:

- використано напіввідкритий інтервал:
  - `>= start`
  - `< end`

Це забезпечує:
- відсутність подвійного підрахунку
- коректну роботу в аналітичних сценаріях
