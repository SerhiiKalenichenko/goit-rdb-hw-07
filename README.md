# goit-rdb-hw-07

Домашнє завдання до теми 7: Додаткові вбудовані SQL-функції. Робота з часом.
---

## Overview
Цей проєкт демонструє:
- використання SQL-функцій для роботи з датою та часом
- виділення року, місяця та дня з атрибута `date`
- додавання інтервалу часу до дати
- перетворення дати у timestamp
- фільтрацію записів за часовим діапазоном
- створення JSON-об’єкта засобами SQL

---

## Task 1 — Extract YEAR, MONTH, DAY

Виконано запит, що:
- отримує дані з таблиці `orders`
- витягує з атрибута `date`:
  - рік
  - місяць
  - день
- відображає:
  - `id`
  - оригінальний `date`
  - `order_year`
  - `order_month`
  - `order_day`

Реалізація:
- використано вбудовані функції:
  - `YEAR()`
  - `MONTH()`
  - `DAY()`

---

## Task 2 — Add 1 Day

Виконано запит, що:
- відображає `id`
- відображає оригінальний атрибут `date`
- додає до `date` один день

Реалізація:
- використано функцію `DATE_ADD(..., INTERVAL 1 DAY)`

---

## Task 3 — Timestamp

Виконано запит, що:
- відображає `id`
- відображає оригінальний атрибут `date`
- показує кількість секунд з початку відліку для значення `date`

Реалізація:
- використано функцію `UNIX_TIMESTAMP()`

---

## Task 4 — Count Rows in Date Range

Виконано запит, що:
- рахує кількість рядків у таблиці `orders`
- залишає тільки записи, де `date` знаходиться в межах:
  - `1996-07-10 00:00:00`
  - `1996-10-08 00:00:00`

Реалізація:
- використано `COUNT(*)`
- застосовано фільтрацію через `BETWEEN`

---

## Task 5 — JSON Object

Виконано запит, що:
- відображає `id`
- відображає `date`
- формує JSON-об’єкт виду:
  - `{"id": ..., "date": ...}`

Реалізація:
- використано функцію `JSON_OBJECT()`

---

## Висновок

Усі 5 завдань виконані відповідно до вимог:
- реалізовано роботу з компонентами дати
- виконано додавання інтервалу часу
- отримано timestamp для значень дати
- виконано підрахунок записів у заданому часовому діапазоні
- сформовано JSON-об’єкти засобами SQL

Рішення є логічним, відтворюваним та готовим до перевірки.
